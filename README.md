## lesson1
знакомство с языком Python 
Сокращения
* iter = 2
* iter += 3 # iter = iter + 3
* iter -= 4 # iter = iter - 4
* iter *= 5 # iter = iter * 5
* iter /= 5 # iter = iter / 5
* iter //= 5 # iter = iter // 5
* iter %= 5 # iter = iter % 5
* iter **= 5 # iter = iter ** 5

Управляющие конструкции: >, >=, <, <=, ==, !=, not, and, or

Range
* r = range(5) # range(0, 5)
* r = range(2, 5) # range(2, 5)
* r = range(-5, 0) # range(-5, 0)
* r = range(1, 10, 2) # range(1, 10, 2)
* r = range(100, 0, -20) # range(100, 0, -20)
